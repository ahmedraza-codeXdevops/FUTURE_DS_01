# 🛒 E-Commerce Customer Behaviour Dashboard

## 📌 Project Overview

This project presents an interactive Power BI dashboard that analyzes customer behaviour on an e-commerce platform. It provides insights into customer interactions, product categories, brands, sales performance, and user activity to support business decision-making.

## 🎯 Objectives

- Analyze customer behaviour on the e-commerce platform.
- Identify the most popular product categories and brands.
- Track customer events such as View, Cart, and Purchase.
- Monitor total users and overall sales.
- Generate business insights for improving customer engagement.

## 📊 Dashboard Features

- Total Users
- Total Sales
- Event Type Distribution
- Brand-wise Analysis
- Category-wise Analysis
- Product Category Distribution
- Customer Activity Overview

## 🛠️ Tools Used

- Microsoft Power BI
- CSV Dataset
- Data Cleaning
- Data Visualization

## 📂 Dataset

- **Dataset:** cleaned_2019_Nov.csv

## 📁 Files Included

## 📈 Key Insights

- Most customer activities are **View** events.
- Samsung and Apple are among the leading brands.
- Electronics categories receive the highest customer engagement.
- The dashboard displays overall sales and total users.
- Customer behaviour patterns help identify business opportunities.

## 🚀 How to Use

1. Download the repository.
2. Open `Ecommerce_Customer_Behaviour_Dashboard.pbix` in Power BI Desktop.
3. Load or refresh the dataset if required.
4. Explore the interactive dashboard.
