# 🇮🇳 Indian E-Commerce Market Analysis Dashboard

## 📌 Project Overview

This project presents an interactive Power BI dashboard that analyzes the Indian e-commerce market. It provides insights into revenue, customer activity, orders, product categories, and platform performance to support data-driven business decisions.

## 🎯 Objectives

- Analyze revenue across major e-commerce platforms.
- Compare platform-wise business performance.
- Track active users and total orders.
- Analyze category-wise revenue distribution.
- Monitor yearly revenue trends.
- Generate actionable business insights.

## 📊 Dashboard Features

- Total Revenue
- Total Orders
- Total Active Users
- Platform-wise Revenue Analysis
- Category-wise Revenue Distribution
- Year-wise Revenue Trend
- Active Users vs Orders Comparison
- Interactive Filters (Platform, Category, Year)

## 🛠️ Tools Used

- Microsoft Power BI
- CSV Dataset
- Power Query
- Data Visualization

## 📂 Dataset

**Dataset:** Indian E-Commerce Market Dataset

## 📁 Files Included

- Indian_Ecommerce_Market_Analysis_Dashboard.pbix
- Indian_Ecommerce_Market_Dataset.csv
- dashboard.png
- README.md

## 📈 Key Insights

- JioMart generated the highest revenue among all platforms.
- Grocery category contributed the largest share of total revenue.
- Revenue trends varied across different years.
- Active users and orders showed a strong positive relationship.
- Platform-wise comparison helps identify top-performing marketplaces.

## 🚀 How to Use

1. Download the repository.
2. Open `Indian_Ecommerce_Market_Analysis_Dashboard.pbix` in Power BI Desktop.
3. Load or refresh the dataset if required.
4. Explore the dashboard using the interactive filters.
